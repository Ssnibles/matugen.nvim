# matugen.nvim

## To get started

### [Lazy.nvim](https://lazy.folke.io/installation)

```lua
{
"ssnibles/matugen.nvim",
    opts = {
        file = "~/.cache/matugen/colors.jsonc",
        plugins = {
            base = true,
            treesitter = true,
            cmp = true,
            lualine = false,
        },
    },
}
```
